**Tenants-HMS - Tenant Management System**
           A simple and efficient Tenant Management System (HMS) built to help landlords and property managers track tenants, lease agreements, and rent payments.
   This project provides an easy-to-use interface for managing tenant details, payments, and lease information

**Features**
  Add new records to the system
  View all saved records
  Edit existing records
  Delete records
  Simple and easy-to-understand code structure
  Beginner-friendly project 

**Technologies Used**
 Frontend : react
 Backend :  Java 
 Database : MySQL
 Version Control : Git & GitHub
